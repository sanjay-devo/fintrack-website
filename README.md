# FinTrack Website

A clean and polished finance tracker built with HTML, Tailwind CSS, and vanilla JavaScript. This lightweight app helps you manage daily income and expense entries, view today’s totals, and keep a running balance stored locally in the browser.

## Features

- Add income and expense transactions quickly
- Track today’s income and expense separately
- View real-time balance updates
- Delete entries with confirmation modal
- Dark/light theme toggle
- Responsive layout for mobile and desktop
- Persistent data using `localStorage`
- Friendly UI with polished spacing, button states, and transaction list styling

## How to Use

1. Open `index.html` in your browser.
2. Click the `Expense` or `Income` buttons to add a new transaction.
3. Enter the amount and optional description.
4. Save the transaction to update your balance and recent activity.
5. Use the theme toggle in the header to switch between light and dark mode.

## Tech Stack

- HTML
- Tailwind CSS
- Vanilla JavaScript

## Notes

- All data is saved locally to the browser using `localStorage`.
- No backend or server setup is required.
- Refreshing the page retains your transaction history.
